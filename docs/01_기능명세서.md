## 1. 제품 개요

**todly**는 가족·룸메이트·커플·친구가 **공동 목표(그룹)** 를 함께 수행하고, **실시간으로 함께하며(라이브 룸)**, **꾸준함을 게임처럼 시각화**하는 협업형 생활관리 앱이다.

핵심 차별점 4가지:
1. **공유 투두** — 그룹/커플/여행/리스트 단위 공동 할 일
2. **그 순간 함께하기** — "내가 할게요" → 라이브 시작 → **라이브 룸**(여러 명 동시 참여 + 사진 공유 + 이모지 응원)
3. **꾸준함 게이미피케이션** — 라이프 점수·루틴 점수·연속일수·**잔디 heatmap**
4. **친구 그래프** — @아이디로 친구 맺고, 친구를 그룹에 초대

## 3. 용어 정의 (Glossary)

| 용어 | 식별자 | 정의 |
|---|---|---|
| 사용자 | User | `@아이디`(handle)·닉네임·프로필색을 가진 계정 |
| 친구 | Friend | 사용자 간 양방향 관계(요청→수락) |
| 그룹 | Group | 공동 목표 단위. **타입**: 그룹/커플/여행/리스트 |
| 투두 | Task | 할 일. 체크리스트·사진·댓글·꾸준함 포함 |
| 섹션 | Section | 그룹 내 투두 묶음(예: 짐 싸기/주소/가구) |
| 라이브 | Live | 투두를 "지금 하는 중"으로 표시한 상태 |
| 라이브 룸 | Live Room | 여러 명이 동시에 참여하는 실시간 방(사진·응원) |
| 루틴 | Routine | 반복 할 일(시간+요일), 연속일수(streak) 추적 |
| 꾸준함 | Consistency | 잔디 heatmap·연속일수·점수 게이미피케이션 |
| 라이프/루틴 점수 | Score | 활동 누적 기반 게임 점수 |
| 테마 | Theme | 오션/민트/바이올렛/코랄/선셋 5색 + 다크모드 |

Scenario: 라이브 룸 사진·응원
